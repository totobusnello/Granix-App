# Arquivo Pré-Pivot — Modelo B2C Standalone

Documentos arquivados em 2026-04-02 durante revisão estratégica do produto.

## Motivo do Arquivo

Esses documentos descrevem o modelo **B2C standalone** original (inspirado no Greenlight EUA):
- Granix como fintech com BaaS próprio (Dock/Pismo/Matera)
- Conta digital própria com cartão físico e virtual
- KYC feito pela Granix
- PIX e transações gerenciados pela Granix

## Novo Modelo

O produto pivotou para **B2B2C SaaS** (ver `technical-spec.md` na raiz):
- Granix como middleware integrado ao banco parceiro
- Banco detém a conta, o cartão, o KYC e o dinheiro
- Granix orquestra comandos e provê a camada de educação financeira
- Potes financeiros são virtuais (caixas contábeis no banco de dados da Granix)
